Selectors

CSS selectors target HTML elements to apply styles.

They include universal, type, class, ID, and attribute selectors.

Advanced selectors like :nth-child and :not() offer precision.

Grouping selectors helps reduce repetition.

Combinators define relationships between elements.

p.intro { color: blue; } div > p { margin: 10px; }
  • Universal selector: *
  • Class selector: .class
  • ID selector: #id
Box Model

Every HTML element is a box with content, padding, border, and margin.

Understanding the box model is key to layout control.

Padding adds space inside the border.

Margin adds space outside the border.

Border wraps the content and padding.

div { margin: 20px; padding: 10px; } box-sizing: border-box;
  • Content
  • Padding
  • Border
  • Margin
Flexbox

Flexbox is a one-dimensional layout model.

It simplifies alignment and spacing of items.

Use `display: flex` to activate it.

Flex-direction controls row or column layout.

Justify-content and align-items manage spacing.

display: flex; justify-content: space-between;
  • Flex container
  • Flex items
  • Direction
  • Alignment
Grid Layout

CSS Grid is a two-dimensional layout system.

It allows precise control over rows and columns.

Use `display: grid` to activate it.

Define grid-template-columns and rows.

Place items using grid-area or line numbers.

grid-template-columns: 1fr 2fr; grid-row: 1 / 3;
  • Grid container
  • Grid items
  • Template areas
  • Line-based placement
Media Queries

Media queries adapt styles to different screen sizes.

They use conditions like min-width or orientation.

Mobile-first design starts with base styles.

Enhancements are layered for larger screens.

Use logical operators like `and` to combine features.

@media (min-width: 768px) { ... } @media (orientation: landscape) { ... }
  • min-width
  • max-width
  • orientation
  • resolution
